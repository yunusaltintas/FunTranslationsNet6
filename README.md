# FunTranslationsAPI 

https://api.funtranslations.com/translate/ 
It is written with the Code First approach.
When you run the project, the database will be created automatically.

Used Technologies:

* .Net6 Core MVC
* RestService
* MSSQL
* Docker
* GrayLog
* Fluent Validation
* AutoMapper
* Rest-Sharp

Best Practices:

* Onion Architecture
* Dependency injection
* Unit of Work
* n-tier Architecture
* DTO
* Generic Repository
* IOptions System Models
